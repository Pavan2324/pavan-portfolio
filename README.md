## 💻 Portfolio Tech

This portfolio is built with **React + TypeScript + Vite + Tailwind CSS** and features an **AI chatbot** powered by Google Gemini that answers recruiter questions about my profile in real time.

### Run Locally

```bash
# Clone the repo
git clone https://github.com/Pavan2324/pavan-portfolio.git
cd pavan-portfolio

# Install dependencies
npm install

# Add your Gemini API key
echo "GEMINI_API_KEY=your_key_here" > .env.local

# Start dev server
npm run dev
```
